Welcome to Glip.

This repository is a fork of the original by Patrik Fimml (@patrikf) and was created by me in an attempt to merge together the various forks that are lying around on GitHub. If you want to contribute in attempting to bring Glip up-to-speed and back to life and then fix outstanding problems, feel free to fork this project and submit pull requests.

I've activated the GitHub wiki and issue pages for this project and set them as open as possible.

Support or Contact

Having trouble? Check out (or help build) the documentation at http://github.com/mvdkleijn/glip/wiki for more information. You can also add (or help solve) issues at http://github.com/mvdkleijn/glip/issues.
